Bagging¶

Ensembling method is used because, to fit models linearly than using a single fit for a method.* Bagging is an algorithm used so that the data can get trained parallelly on different samples. In this data det of each sample will be different and it is based on the error being generated by the last training, will be still using the same data (this will be the residual of the last training set).
